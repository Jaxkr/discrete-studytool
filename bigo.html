Definitions

  • Big O - A function that represents the maximum possible value for a function (upper bound).
  • Big Ω - A function that represents the minimum possible value for a function (lower bound).
  • Big Θ - A function which represents the function behavior, and is simply equal to Big O (or Big Ω) if any only if Big O and Big Ω are the same.

An example function

Suppose we want to find the Big O notation for the following function:

$$f(n) = 5n^2$$

This is a very simple quadratic function, and it's Big O upper bound is $O(n^2)$.
Due to its simplicity, it's Big Ω lower bound is $Ω(n^2)$.

Quick comprehension check

Consider again our previous function $f(n) = 5n^2$ with $O(n^2)$ and $Ω(n^2)$

What is Big Θ? Remember that Big Θ is only defined if Big O and Big Ω are equal.


Correct! Since the Big O and Big Ω for this function are equal, big Θ is defined for this function and is equal to Big O (or Big Ω).

Solving for a Big O estimate

Consider this new function $f(n) = 5n^2 + 10log(n)$. The correct way to approach this problem is through a series of inequalities.

$ |f(n)| = |5n^2 + 10log(n)| $
It's true that $log(n) \leq n$ for all values $n > 1$
$|f(n)| \leq |5n^2 + 10n|$
$|f(n)| \leq |5n^2 + 10n^2|$
$|f(n)| \leq |15n^2|$
To get the big O bound, we take the function without any coefficients. Therefore, this function is:
$O(n^2)$ with $c=15$ and $k=1$
The c value is the coeffient in front of the $n^2$, and k is the minimum value for which our inequalities hold (which we defined above as 1).
